# Single Key Shortcuts

A WebExtension that provides a few hardcoded old-Opera style shortcuts.

1. z: history back
1. x: history forward
1. c: previous tab
1. v: next tab

## Building

The configuration page uses vue.js.

Install with:

`npm install`

Build with:

`npm run build`

This generates `dist/build.js`

## Packaging

Package with:

`./package.sh`
